TerraGenitor:get_map (x, z, support)
Gets the map for the given x and z coordinate. The map is a 2D array/table, with the first dimension being x, the second being z starting at the given values. The values contained within is a table with two values, value and info. Both are not defined. The size of the map defaults to constants.block_size, which should be 80. Example: value = map[x][z] The result of this function is cached, and if invoked multiple times the map from the cache is returned.

Parameters

  • x: The x coordinate.
  • z: The z coordinate.
  • support: Optional. The support object that provides additional additional information for the module. The format of this is not defined.

Return value:

A map with all values starting at the given coordinates.
