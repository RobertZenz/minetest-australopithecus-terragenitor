Caelum:get_biome_map (x, z, elevation_map)
Gets the map of the biomes for the given x and z coordinate. The map is a 2D array of biomes with the first dimension being x, and the second being z, starting at the given coordinates.

Parameters

  • x: The x coordinate.
  • z: The z coordinate.
  • elevation_map: The 2D array that contains the elevation values for the given x and z coordinates.

Return value:

A map with the biomes starting at the given coordinates.
