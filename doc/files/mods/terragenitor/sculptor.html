Sculptor:init (noise_manager)
Inits this instance and all chisels. Does nothing if this instance is already intialized.

Parameters

  • noise_manager: Optional. The NoiseManager to use for the init. Defaults to a new instance if not provided.
Sculptor:is_initialized ()
Gets if this instance has been initialized.

true if this instance has been initialized.
Sculptor:new ()
Creates a new instance of Sculptor.
Sculptor:register (chisel)
Register the given chisel. Chisels should be registered before the init method is called, otherwise the chisels will not be initialized by Sculptor.

Parameters

  • chisel: The chisel to register.
Sculptor:sculpt (x, y, z, data, area, support)
Sculpts the given data by calling all chisels on it.

Parameters

  • x: The x coordinate.
  • y: The y coordinate.
  • z: The z coordinate.
  • data: The data which should be changed.
  • area: The area associated with the given data.
  • support: Optional. The support object that provides additional information for the chisels. The format is not defined.

Return value:

The same data object as given.
