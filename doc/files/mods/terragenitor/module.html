Module:get (x, z, value, info, support)
Gets the value and info from this Module. The info object is an undefined table, which allows each module to save individual information.

Parameters

  • x: The x coordinate.
  • z: The z coordinate.
  • value: The base value.
  • info: The info object.
  • support: Optional. The support object that provides additional additional information for the module. The format of this is not defined.

Return value:

The value and the info object.
Module:init (noise_manager)
The init method is called when TerraGenitor's init method is called. It is only called once right at the beginning.

Parameters

  • noise_manager: The NoiseManager that is used.
Module:init_map (x, z)
The init_map method is called before values for a certain segment of the map are requested.

Parameters
